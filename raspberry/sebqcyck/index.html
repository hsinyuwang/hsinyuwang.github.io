树莓派安装docker

Jul 30, 2021
1 min read
树莓派

树莓派最常用的系统为Raspbian，是基于Debian的一套Linux系统。查询docker官网可以看到这样一句提示：

Raspbian users cannot use this method!

For Raspbian, installing using the repository is not yet supported. You must instead use the convenience script.

所以根据官网建议，我们最好是使用docker官方提供的便捷脚本来一键安装

$ curl -fsSL https://get.docker.com -o get-docker.sh
$ sudo sh get-docker.sh
